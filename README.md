TruSwap is a robust, full-stack marketplace developed for Truman University students to securely buy, sell, and rent items. The project uses a modern, containerized architecture and won First Place in the Fall 2025 ACM Codeathon.

Key Features & Impact

    Secure Transactions: Integrated Auth0 for authentication and PayPal for secure payments.

    Flexible Listings: Supports buying, selling, and renting, including unique group-based listings.

    Cloud Deployment: Fully Dockerized and deployed on Google Cloud Run (Backend) and Vercel (React Frontend).

    Tech Stack: Built on Spring Boot (Java API), React (Frontend), and MongoDB (Database).
